# 概要
本アプリケーションは、プログラミング講師や教育機関を対象に、効率的で効果的なPythonプログラミング学習をサポートするために設計されています。管理者は教材やレッスンを作成し、ユーザー（学習者）に割り当て、進捗を追跡することができます。学習者は段階的に課題を解決しながらスキルを向上させることが可能です。
- Kidz8

# 採択技術
- フロントエンド：React
- バックエンド: Python（Flask）
